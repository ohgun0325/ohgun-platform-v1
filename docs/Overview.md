# KOICA AI 플랫폼 - 쉽게 이해하는 프로젝트 개요

## 📌 이 문서는?

이 문서는 코딩 용어를 잘 모르는 분들도 KOICA AI 프로젝트가 **무엇을 하는지, 왜 중요한지, 어떤 혜택이 있는지**를 쉽게 이해할 수 있도록 작성되었습니다.

## 🎯 프로젝트 한 줄 요약

**"KOICA의 입찰·문서 업무를 AI가 자동으로 도와주는 똑똑한 비서 시스템"**

---

## 💡 왜 이 프로젝트가 필요한가요?

### 현재 KOICA 업무의 어려움

1. **방대한 자료 검색의 어려움**
   - KOICA는 수천 건의 보도자료, 사업 보고서, 용어사전, 규정 문서를 보유
   - 필요한 정보를 찾는데 많은 시간이 소요됨
   - 직원들이 "이 사업은 어느 나라에서 했었지?", "이 용어의 정확한 뜻은?" 같은 질문에 답을 찾기 위해 여러 문서를 뒤져야 함

2. **입찰 서류 검토의 비효율**
   - 입찰업체가 제출한 제안서를 하나하나 검토하는데 며칠이 걸림
   - 인감도장이 있는지, 서명이 누락되지 않았는지 일일이 확인해야 함
   - 제안서가 입찰공고(RfP) 요구사항을 잘 따랐는지 대조하는 작업이 매우 복잡함

3. **문서 작성의 반복 작업**
   - 비슷한 내용의 제안서나 보고서를 매번 새로 작성
   - 정해진 용어를 정확하게 사용했는지 확인하는 작업 필요

---

## 🚀 우리 프로젝트가 제공하는 해결책

### 1. **AI 비서 - 질문하면 바로 답해줍니다**

**사용 사례:**
- 직원: "탄자니아에서 진행한 보건 사업이 뭐가 있지?"
- AI: "탄자니아 보건사업은 총 3건입니다. 2023년 모자보건 개선사업, 2024년..."

**어떻게 가능한가요?**
- KOICA의 모든 문서를 AI가 읽고 학습했습니다
- 직원이 일상 대화처럼 질문하면, AI가 관련된 정보를 찾아서 정리해서 알려줍니다
- 검색엔진보다 똑똑합니다 - 정확히 같은 단어가 아니어도 의미를 이해하고 찾아줍니다

### 2. **자동 문서 검증 - 서류 누락을 즉시 발견합니다**

**사용 사례:**
- 입찰업체가 제안서를 제출하면, AI가 자동으로 확인합니다
- "사업예산서가 빠졌습니다", "3페이지에 법인 도장이 없습니다" 같은 문제를 즉시 알려줍니다

**어떻게 가능한가요?**
- AI가 PDF 문서를 읽고, 이미지를 분석해서 도장과 서명을 찾아냅니다
- 필수 서류가 모두 있는지 자동으로 체크합니다
- 사람이 며칠 걸릴 일을 몇 분 안에 완료합니다

### 3. **제안서 초안 자동 작성 - 문서 작성을 빠르게 시작합니다**

**사용 사례:**
- 직원: "베트남 교육사업 제안서 초안을 만들어줘"
- AI: "다음과 같은 구조로 초안을 작성했습니다... (사업 배경, 목표, 예산 등)"

**어떻게 가능한가요?**
- 과거 KOICA가 작성한 수백 개의 제안서를 AI가 학습했습니다
- 새로운 제안서의 기본 틀과 내용을 자동으로 생성해줍니다
- 직원은 AI가 만든 초안을 수정하기만 하면 되어 시간이 크게 절약됩니다

### 4. **요구사항 자동 대조 - 제안서 평가를 정확하게 합니다**

**사용 사례:**
- 입찰공고에 "20명 이상 역량강화 워크숍 개최"라는 요구사항이 있었습니다
- AI가 제안서를 읽고 "17페이지에 30명 대상 워크숍 계획이 있어 요구사항을 충족합니다"라고 자동 평가합니다

**어떻게 가능한가요?**
- AI가 입찰공고의 요구사항을 하나하나 추출합니다
- 제안서를 읽고 각 요구사항이 충족되었는지 자동으로 판단합니다
- 평가 보고서를 자동으로 만들어 줍니다

## 🧩 시스템 구성 요소 (쉽게 이해하기)

프로젝트는 크게 **3가지 부품**으로 구성됩니다:

### 1. **두뇌 (AI 모델)**
- **EXAONE**: 한국어를 이해하고 답변을 생성하는 AI (ChatGPT라고 생각하시면 됩니다)
- **KoELECTRA**: 문서를 읽고 분류·판단하는 AI (검증 전문가)
- 이 두 AI가 협력해서 업무를 처리합니다

### 2. **기억 장치 (데이터베이스)**
- KOICA의 모든 문서, 사업 정보, 용어사전이 저장됩니다
- AI가 질문을 받으면 이 저장소에서 관련 정보를 빠르게 찾습니다
- "의미 검색" 기능으로 똑같은 단어가 아니어도 비슷한 내용을 찾아냅니다

### 3. **인터페이스 (사용자 화면)**
- 직원들이 쉽게 사용할 수 있는 웹 페이지
- 채팅하듯이 질문을 입력하면 AI가 답변
- PDF 파일을 업로드하면 자동으로 분석
